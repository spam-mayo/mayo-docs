## 스터디 관리

### 회원

- 회원가입 / 로그인 / 로그아웃 / pw 찾기
- 마이페이지
- 회원가입
    - 이메일(id) / 패스워드 / 닉네임(활동)
    - 활동 분야 (프론트 / 백 / 디자이너 / 기획)
    - 이메일 인증
        - 가입 중 인증 / 가입 후 인증은 추후 진행하면서
    - 회원가입 이후 로그인 창으로 리다이렉팅
- 로그인 / 로그아웃
    - 이메일 / 패스워드
    - jwt 토큰 / 세션 (redis) 추후 협의
- pw 찾기
    - 이메일 입력
        - 비밀번호 변경 창 링크

### 스터디 생성

- 스터디 생성
    - 이름 / 목적 / 일정 / 주기 / 인원 / 규칙
    - 온라인 / 오프라인 구분
- 스터디 만들기 (버튼)
    - 스터디 명을 입력할 수 있음
        - 중복 가능
    - 목적을 입력할 수 있음 (100자)
    - 일정을 선택할 수 있음
        - 시작 일자 선택
        - 종료 일자 선택
        - 며칠인 지를 계산해서 표시
    - 주기
        - 드랍 박스 + 입력창
            - (매월) 며칠
            - (월 중) 몇 회
            - (매주) 무슨 요일
            - (주 중) 몇 회
    - 인원
        - 운영 인원
    - 규칙 (디자이너 분께 떠넘기기)
        - 규칙을 입력할 수 있음 (100자)
        - (+) 버튼
            - 한 줄 씩 입력
    - 온라인 / 오프라인 선택할 수 있음
- 스터디를 생성하면 스터디 페이지가 생기고, 내부적으로 스터디 일정 탭, 인원관리 탭 등이 생성된다.
- 스터디 해체
    - 스터디원이 없을 경우 해체 버튼을 통해 모달창을 띄워 재확인 후 스터디를 폐쇄할 수 있음.
    - 스터디원이 있으면 50% 이상 인원의 동의가 있어야 함
        - 개별 메시지? 어떤 형태로 찬반 투표 할지
        - ex) 스터디원에게 폐쇄 찬반 투표 메시지를 보낼 수 있다. (쪽지 기능?)
        - ex) 페이지 접속하면 폐쇄 찬반 투표하는 섹션이 추가되어 있다.

### 스터디 구인

- 스터디원을 구하는 글을 작성
    - 스터디 구인 글에서 댓글을 통한 문의를 진행할 수 있음
- 스터디에 지원할 수 있다. / 스터디장이 참가 여부를 결정할 수 있다.
    - 내가 지원한 스터디 응답 결과를 받아볼 수 있음 (승인 / 대기 / 거부)
- 구인버튼 → 구인 중 / 구인 완료 등의 상태 전환이 가능하도록 처리
- 구인글이 생성 (스터디 생성할 때 작성한 정보를 기반으로 구인글 페이지 생성)
    - 댓글 기능을 통해 문의를 받을 수 있음
    - 비밀 댓글 여부 선택 가능 (boolean) → 어려우면 빼기
- 구인 중에는 스터디 참가 신청을 받을 수 있음
- 스터디 인원 관리 페이지가 존재
    - 내부적으로 신청 관리 탭이 존재
    - 스터디 장은 (대기) 참가 신청에 대해 승인 / 거부 처리를 할 수 있음

### 스터디 인원 관리

- 스터디 탈퇴 (자의적) / 스터디 추방 (스터디장)
- 스터디 권한이 존재
    - 스터디 장은 탈퇴가 불가
    - 스터디 장이 탈퇴하기 위해선 특정 회원에게 스터디 장 권한을 양도
- 스터디 해체 가능 (XX% 이상 인원의 동의)
- 스터디원은 탈퇴버튼을 통해 탈퇴할 수 있음
- 스터디장은 스터디 인원 관리 탭에서 추방버튼을 통해 스터디원을 추방할 수 있음
- 스터디장은 스터디원이 존재하면 스터디를 탈퇴할 수 없음
- 스터디장은 스터디를 해체하기 위해선 50% 이상 인원의 동의가 있어야 함

### 스터디 일정 관리

- 스터디 일정이 존재
    - 스터디원은 스터디 구인글을 눌러서 일정 관리로 들어오는지?
    - or 아니면 스터디 관리 페이지가 따로 있어야 할지?
- 달력에 표시
    - 일정을 선택하면 세부 활동 내역이 표시 (editor)
- 온라인 / 오프라인 스터디
    - 오프라인 스터디 장소 표시
- 노션 → 달력 기능 베끼기(?)
- 달력에서 날짜를 선택하면 세부 스터디 일정을 작성 및 편집 할 수 있음
    - 몇 시부터 몇 시까지 진행
    - 스터디 진행 내용 (텍스트 100자)
    - 온라인 / 오프라인 표시 가능
- 세부 일정이 작성된 날짜는 달력 상에 별도 표시
